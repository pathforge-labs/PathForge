# /help Command

Show available commands and usage.

## Usage

```
/help                  # List all commands
/help <command>        # Detail on specific command
```

## Examples

```
/help
/help plan
/help verify
```

- **Core**: /plan, /implement, /verify, /status
- **Development**: /build, /fix, /debug, /refactor
