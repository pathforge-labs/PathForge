# /setup Command

Initialize or configure the AI kit for a project.

## Usage

```
/setup                 # Full setup
/setup <component>     # Setup specific component
```

## Examples

```
/setup
/setup TypeScript
/setup Git hooks
```

## Setup Items

- Project structure
- TypeScript configuration
- ESLint/Prettier
- Git hooks
- CI/CD templates
- Testing setup
