## ğŸ“‹ Commit Message

Follow conventional commits format:

```

[optional body]

[optional footer(s)]
```

**Types**: `feat`, `fix`, `docs`, `style`, `refactor`, `test`, `chore`, `perf`, `ci`

**Examples**:

```bash

git commit -m "fix(api): handle null user in profile endpoint"
git commit -m "docs(readme): add installation instructions"
```
